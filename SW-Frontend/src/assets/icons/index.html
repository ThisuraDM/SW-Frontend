Usage

Prisma Icons are SVGs, so you can include them into your HTML in a few ways depending on how your project is setup.

Sprite

Use the SVG sprite to insert any icon through the <use> element. Use the icons filename as the fragment identifier (e.g., toggles is #toggles). SVG sprites allow you to reference an external file similar to an <img> element, but with the power of currentColor for easy theming. Sprites are currently limited to normal icons, colored icons and expressive icons only.

<svg class="icon" aria-hidden="true" width="32" height="32">
  <use xlink:href="./sprite/prisma-icons.symbol.svg#24-hours"></use>
</svg>

<svg class="icon" aria-hidden="true" width="32" height="32">
  <use xlink:href="./sprite/prisma-icons.symbol.svg#health"></use>
</svg>

<svg class="icon" aria-hidden="true" width="32" height="32">
  <use xlink:href="./sprite/prisma-icons.symbol.svg#home-dashboard"></use>
</svg>

Image

Copy the Prisma icons SVGs to your directory of choice and reference them like normal images with the <img> element.

<img src="./svg/logo/logo-SW-life.svg" alt="" width="32" height="32" title="SW Life">

Styling

Color can be changed by setting a .text-* class or custom CSS. Current usage is limited only for normal icons.

<svg class="icon text-primary" aria-hidden="true" width="32" height="32">
  <use xlink:href="./sprite/prisma-icons.symbol.svg#on-off-power"></use>
</svg>

<svg class="icon" aria-hidden="true" width="32" height="32" style="color: #f44236;">
  <use xlink:href="./sprite/prisma-icons.symbol.svg#love"></use>
</svg>
